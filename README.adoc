== Build

To build the debug code, create an out of source cmake build with

[.bash]
----
cmake -B _build_debug -G Ninja -DCMAKE_BUILD_TYPE=Debug
cd _build_debug
ninja
----

To compile the pybind11 module

[.bash]
----
cmake -B _build -G Ninja -DCMAKE_BUILD_TYPE=Release
cd _build
ninja
----
